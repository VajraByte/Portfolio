## Color Palette

### Primary Colors
- **Navy Blue (Primary):** `oklch(0.25 0.15 250)` 
  - RGB: `#1a3a5c` (approximately)
  - Usage: Primary buttons, main CTAs, brand accents
  
- **Saffron Orange (Secondary):** `oklch(0.62 0.25 45)`
  - RGB: `#e8a832` (approximately)
  - Usage: Highlights, hover states, key information
  
- **Gold (Accent):** `oklch(0.55 0.2 55)`
  - RGB: `#d4a156` (approximately)
  - Usage: Accent elements, gradients, premium feel

### Neutral Colors
- **Background:** `oklch(0.08 0 0)` - Deep dark (almost black)
- **Card Background:** `oklch(0.12 0 0)` - Slightly lighter dark
- **Foreground/Text:** `oklch(0.95 0 0)` - Off-white
- **Border:** `oklch(0.2 0 0)` - Subtle dark border
- **Muted Text:** `oklch(0.7 0 0)` - Gray text

## Typography
- **Font Family:** System UI / -apple-system / sans-serif
- **Headings:** Bold weight (600-700), Navy or Saffron color
- **Body Text:** Regular weight (400), Off-white foreground
- **Line Height:** 1.4-1.6 for readability

## Design Elements
- **Logo:** "V" lettermark in gradient (Navy → Saffron) within rounded square
- **Border Radius:** 0.625rem (10px) for consistency
- **Shadows:** Secondary color shadows for depth (e.g., `shadow-secondary/10`)
- **Gradients:** Navy → Saffron, Saffron → Gold for premium feel

- Maintain consistent spacing using the 8px grid system
- Use the gradient sparingly for premium brand moments
